> **Emby常见HTTP 标准错误码，如遇错误代码请自查自测**


## 🔴 **4**xx（客户端问题）
- **400** 请求参数错误
- **401** 未提供认证信息
- **403** 已认证但无权限
- **404** 资源不存在
- **405** 不支持的请求方法

## 🟢**5**xx（服务端问题）
- **500** Emby 内部临时错误
- **501** 功能未实现
- **502** 网关无法连接 Emby
- **503** 正在重启或过载
- **504** 后端响应超时
- **505** HTTP 版本不支持

## 🤍 Cloudflare 错误码（套 CF 时常见）
- **520** 源站返回未知响应
- **521** 源站宕机
- **522** 连接源站超时
- **523** 无法到达源站（DNS）
- **524** 源站响应超时（CF 侧）
- **525** SSL 握手失败
- **526** SSL 证书无效
- **530** 源站 DNS 解析错误
