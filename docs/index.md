## 安全码说明
>安全码用于 注销账号、重置密码、换绑TG 等敏感操作。

- **正常用户：** `安全码 = 你的 Emby 登录密码 + syemby`,例：`密码是 abc123 → 安全码是 abc123syemby`。
- **密码错误的用户（原有的emby账号密码无法登录emby）:** `安全码 = 你的 Emby 用户名 + syemby99`,例：`用户名是 zhangsan → 安全码是 zhangsansyemby99`

## 修改密码步骤
1. 私聊 [@syembyy_bot](https://t.me/syembyy_bot)
2. 发送 /start 打开面板
